In the Style tab of the Layers Properties Dialog, click Simple fill. Then, change some settings at your will. For example:

  • Set the Fill color to yellow
  • Set Border color to Red
  • Set Border width to 1.0

When your are happy with the new style, click Apply to check the changes in the map canvas.

To make this style the default, click Style at the bottom of the dialog and choose Save as Default

Click OK to apply the changes and close the Layers Properties Dialog.

Note: You could also change other settings in the layer's properties, like enabling Labels, adding Diagrams, setting custom widget in Fields and so on. Everything is saved in the default style.

Click Next step once you are done.
