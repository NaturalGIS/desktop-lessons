In the Add Vector layer dialog, click Browse.

In the Open an OGR Supported Vector Layer dialog, make sure you have the file type filtered to "All files".

From your user's home folder, and go to the qgislessonsdata\save_default_style folder and select the Wake_BlockGroup_2010.zip file. Then, click Open.

Back in the Add Vector Layer, press Open to load the layer.

Click Next step once you are done.
