As you can see, the raster is now styled using a more diverse range of color, going from red (low values) to green (high values), passing by yellow.

You may notice two other things:

  1. The values in the legend are not very user-friendly.
  2. We used a linear color interpolation, which means that values between the legend's values draw in the map canvas with intermediate color tones.
