In the Style tab of the Layer Properties, click the Expression button next to Column.

In the Expression dialog, type the following expression:

"CLASSNAME" + ', ' + "ONE_WAY"

The + (plus) sign will concatenate the text values of both fields. We also added a text separator, composed of a colon and a space. Click OK.

Back in the Layer properties dialog, click Classify. When asked if you want to delete existing classes, click Yes.

Click OK to apply the changes and close the Properties dialog.
