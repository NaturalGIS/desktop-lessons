In the Style tab of the Layer Properties, click the Expression button next to Column.

In the Expression dialog, change the expression to the following:

"CLASSNAME" + ', ' + replace("ONEWAY", 'FT', 'TF')

In this expression, we use the replace() function to transform any FT value in a TF value. Click OK.

Back in the Layer properties dialog, click Classify. When asked if you want to delete existing classes, click Yes.

Click OK to apply the changes and close the Layer properties dialog.
