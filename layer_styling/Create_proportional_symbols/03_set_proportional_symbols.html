In the Layer Properties dialog:

  • Click the Style tab;
  • Click the data-override button next to Size and choose Size assistant.

In the Size assistant dialog:

  • For Field select the capacityto field;
  • Set Scale method to Flannery;
  • Set Size from 2.0 to 10.0;
  • Set Values From 0.0 to 2500.0;
  • Click OK

Click OK to apply the changes and close the Layer Properties dialog.
