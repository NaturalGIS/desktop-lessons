In the Style tab of the Layer Properties dialog, change the symbol type, from single symbol to Rule-based.

Notice that QGIS will convert the previous symbol to a rule with no filter (draw all). Double-click the rule to open its properties.

In the Rule properties dialog:

  1. Set Label to Homes.
  2. Set Filter to "DESCRIPTIO" IN ( 'RESIDENTIAL' , 'MOBILE HOME' )
  3. Set the Symbol's Color to a brighter color, like Orange.
  4. Click OK

Back in the Layer Properties dialog, click OK to apply the changes and close the dialog.
