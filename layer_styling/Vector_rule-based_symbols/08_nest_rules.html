Because we used the ELSE rule, before we can create another unrelated rule, we need to nest the previous two rules together with a "blank" rule.

In the Style tab of the Layer Properties dialog, click the Add rule button.

In the Rule properties dialog:

  1. Set Label to Type
  2. Leave the Filter empty
  3. Deselect the checkbox next to Symbol
  4. Click OK

Back in the Layer Properties dialog, click the two other rules while holding the Ctrl key to select them. Then, drag and drop them on top of the newly created rule.

Click OK to apply the changes and close the Layer properties dialog.
