In the Style tab of the Layer Properties, click the Expression button next to Column.

In the Expression dialog, type the following expression:

("TOTAL_POP" / $area) * 5280 ^ 2

One mile is 5280 feet, so multiplying the previous values of population density by the square of 5280 will return the number of inhabitants by square mile.

If we wanted to use square kilometers, we could multiply the conversion factor by 1.60934

("TOTAL_POP" / $area) * (5280 * 1.60934) ^ 2

Click OK.

Back in the Layer properties dialog, click Classify.

Click OK to apply the changes and close the Layer properties dialog.
