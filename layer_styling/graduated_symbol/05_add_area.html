In the Style tab of the Layer Properties, click the Expression button next to Column.

In the Expression dialog, enter the following expression:

"TOTAL_POP" / $area

The $area function returns the size of each feature. By dividing the TOTAL_POP value by the area, we get a population density variable. Click OK.

Back in the Layer properties dialog, in the Classes tab, click Classify.

Click OK to apply the changes and close the Layer properties dialog.
