In the Style tab of the Layer Properties, click the Expression button next to Column.

In the Expression dialog, enter the following expression:

"OCCUPIED" / "TOTAL_HU" * 100

Click OK

Back in the Layer properties dialog:

  1. Set the Legend Format to %1 - %2 %
  2. Set Precision to 0
  3. Change the Color ramp to, for example, Oranges;
  4. In the Classes tab, set Classes to 4
  5. Click Classify.

Click OK to apply the changes and close the Layer properties dialog.
