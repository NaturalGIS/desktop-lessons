In the Layer Properties dialog:

  • Click the Labels tab;
  • Change from No labels to Show labels for this layer;
  • For Label With select the ZIPCODE field;
  • Click the Text options tab and:
    • Set Style as Bold Italic;
    • Set Size as 10;
    • Set Color as brown;
  • Click the Buffer options tab and:
    • Enable the Draw text buffer;
    • Set Size to 1.0.

For now, leave all other settings with the default values.

Click OK to apply the changes and close the Layer properties dialog.
