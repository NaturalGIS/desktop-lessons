In the Labels tab of the Layer Properties Dialog:

  • Change from No labels to Show labels for this layer;
  • For Label With select the cartoname field;
  • Click the Text options tab and:
    • Set Size as 10;
    • Set Color as dark green;
  • Click the Buffer options tab and:
    • Enable the Draw text buffer;
    • Set size to 0.5;
  • Click the Placement options tab and:
    • Select Curved;
    • For Allowed positions choose only On line;

Click OK to apply the changes and close the Layer properties dialog.
