In Labels tab of the Layer properties dialog:

  • Click the Formatting options tab;
    • In the Wrap on character option type a simple "space";
    • Set Alignment to Center;
  • Click the Placement options tab and:
    • For Placement choose Horizontal (slow);
  • Click the Rendering options tab and:
    • Scroll down to the Feature options and select the Only draw labels which fit completely within feature.

Click OK to apply the changes and close the Layer properties dialog.
