In the Style tab of the Layer Properties dialog, click the Add rule button.

In the Rule properties dialog:

  1. Set Label to Outdated
  2. Set Filter to year("UPDATE_DAT") < 2010
  3. In the Symbol options, click Simple fill, then:
    • Set the Symbol layer type to Outline: simple line
    • Set Color to a strong color like Red
    • Set Pen width to 1
  4. Click OK

Back in the Layer Properties dialog, click OK to apply the changes and close the dialog.
