In the Style tab fo the Layer Properties, notice that you can add and remove new value entries in the values list. You can also manually change the each entry's Value, Color or Label by double-clicking their values. We will do it automatically instead.

In the Generate new color map options:

  1. Set the mode to equal interval
  2. Keep the number of Classes as 5
  3. Set the Min to 500
  4. Set Max to 2500
  5. Click Classify

Click OK to apply the changes and close the Layer properties dialog
