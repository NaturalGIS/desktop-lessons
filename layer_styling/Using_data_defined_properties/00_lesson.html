Using data-defined properties

In QGIS, most styling and labelling options of vector layers have the ability to use data-override values. This means that you can use the layer attributes' values to set it them. The possible use cases for this feature are endless. This lesson presents a couple of simple examples on how to use data-defined properties to control a vector layer's style.
