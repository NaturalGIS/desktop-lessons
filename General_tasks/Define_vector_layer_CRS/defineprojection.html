In the Define current projection dialog:

  1. For Input vector layer choose the Wake_BlockGroup_2010 layer.
  2. Under Output spatial reference system choose Use predefined spatial reference system and click Choose.
  3. In the Coordinate Reference System Selector dialog, type North Carolina in the Filter field and select the NAD83/North Carolina (ftUS) | EPSG:2264 in the Coordinate Reference Systems of the world list. Click OK
  4. Back at the main dialog, click OK to apply and close the dialog.

The CRS is saved on the original data, and the layer will no longer load into QGIS with a wrong CRS.

Once you have finished, click Next step, to end the lesson.
