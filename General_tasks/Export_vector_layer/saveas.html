In the Save as vector layer as dialog, do:

  1. Select GeoJSON as the Format.
  2. Type directly on the Save as field or use the Browse button to select the destination and filename of the file to create. Set the filename as points.geojson

  3. Enable the Add saved file to map option.
  4. Click OK to create the new layer.

The new layer will load in the map canvas.

Once you are done, click Next step to finish the lesson.
