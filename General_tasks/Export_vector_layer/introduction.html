One of the more common GIS tasks is converting data over different formats. QGIS can read tens of vector formats and then convert them back to the most common ones. Most of the times, all you need to use is the simple "Save as" tool over a loaded layer.

This lesson's project contains a layer named Wake_Fire_Stations, which source's format is ESRI shapefile. In this lesson, we will export it to the GeoJSON format.
