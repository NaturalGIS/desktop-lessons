We saw how to fix the layer's CRS by setting it correctly within the project. But, fixing it this way, the problem will happen every time you load the layer in a project.

To fix the CRS problem once and for all, we need to solve it at the data level. That is, change the data's CRS definition permanently.

  1. Click on Vector > Data Management Tools > Define Current Projection...' menu item.

Once you click it, the lesson will automatically move to the next step.
