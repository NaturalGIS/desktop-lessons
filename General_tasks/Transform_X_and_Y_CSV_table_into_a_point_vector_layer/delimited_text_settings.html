In the Create a Layer from a Delimited Text File dialog, do:

  1. Click Browse.
  2. From your user's home folder, and go to the qgislessonsdata\Transform_X_and_Y_CSV_table_into_a_point_vector_layer folder, select the songs.csv file, and click Open.
  3. Back at the dialog, set the options as follows:
    • Layer name = songs_in_madrid
    • File format = Custom delimiters
    • Enable the Semicolon option
    • In record options, Enable First record has field names
    • Geometry definition = Point coordinates
    • X field = X
    • Y field = Y
  4. Press OK to proceed and close the dialog

A dialog will ask you the layer's CRS, choose WGS84 EPSG:4326.
