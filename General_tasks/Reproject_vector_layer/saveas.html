In the Save as vector layer as dialog, do:

  1. Select ESRI Shapefile as the Format.
  2. Type directly on the Save as field or use the Browse button to select the destination and filename of the file to create. Set the filename as Wake_Fire_Stations_WGS84.geojson.
  3. In the CRS field, select the destination CRS. The Wake_Fire_Stations layer is in the EPSG:2264 CRS. Set it to EPSG:4326 either using the drop-down menu or the Select CRS button to the right.

  4. Enable the Add saved file to map option.
  5. Click OK to create the new layer.

The new reprojected layer will load in the map canvas.

This step concludes the lessons. click Finish to end the lesson.
