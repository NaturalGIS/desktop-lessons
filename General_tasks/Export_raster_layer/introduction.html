One of the more common GIS tasks is converting data over different formats. QGIS can read tens of raster formats and then convert them back to the most common ones. For that purpose, all you need to use is the GDAL Translate (Convert format) tool.

This lesson's project contains a layer named dem25, which source's format is a GeoTiff. In this lesson, we will export it to the ArcInfo ASCII format.
