In the previous lesson, we have seen how to reproject a vector layer using the save as tool from the Vector menu. In this lesson, we will use the processing Reproject layer tool instead.

Although in this case, the tools' outputs are the same, in general, processing tools ofter extra functionality. In this particular case:

  • the tool can be chained with other processing tools in a model
  • You can run the tool in batch process, which means you can select and reproject several layers in one go.
  • It's possible to reproject to a temporary layer
  • For some settings you can use expression builder to choose the value
