In the Grid Decoration dialog, do the following:

  1. Click the checkbox to Enable gridr.
  2. Set Interval X as 100000
  3. Set Interval Y as 100000
  4. Keep Grid type as Line.
  5. Click the checkbox Draw annotation to enable it.
  6. Set Annotation direction as Boundary direction
  7. Keep the Distance to map frame as 2.
  8. Keep Coordinate precision as 0
  9. Keep Set direction automatically enabled
  10. Click OK to apply the settings and close the dialog

A grid will appear on map canvas, showing labels with the X and Y coordinates next to it's lines.
