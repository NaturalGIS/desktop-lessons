In the Layer Properties dialog, do:

  1. Go to the General tab.
  2. Under Coordinate reference system make sure that the Selected CRS is EPSG:23030, ED50 /UTM zone 30N

  3. If it is not, select the correct CRS for your layer, either using the combobox or the Select CRS button to the right.
  4. Click OK to apply the changes and close the dialog.

Once you are done, click Move to next step to continue the lesson.
