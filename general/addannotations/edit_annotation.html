Let's edit our the annotation text and settings. For that, Double-click inside the annotation box.

In the Annotation text dialog, do:

  1. Type some text in the text field. For example, "This is a nice place to put an annotation"
  2. To change the font type, size or color, you need to select part of text. In this case, select it all.
  3. Font size = 12
  4. Frame width = 1.5
  5. Background color = yellow
  6. Click OK to apply the changes and close the dialog
