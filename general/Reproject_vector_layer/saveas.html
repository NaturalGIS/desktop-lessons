In the Save as vector layer as dialog, do:

  1. Select ESRI Shapefile as the Format.
  2. Type directly on the Save as field or use the Browse button to select the destination and filename of the file to create.
  3. In the CRS field, select the destination CRS. The points layer in the project is in the EPSG:23030 CRS. Set it to EPSG:23029.

  4. Enable the Add saved file to map option.
  5. Click OK to create the new layer.

The new reprojected layer will load in the map canvas.

Once you are done, click Move to next step to finish the lesson.
