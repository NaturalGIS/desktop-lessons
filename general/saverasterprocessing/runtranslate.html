In the Translate (Convert Format) algorithm dialog, do:

  1. Set the Input layer to be the dem25 layer.

  2. Type in directly or use the '...' button next to Converted to choose the location and filename of the file to be created.

    3. Note: The filename extension determines its output format. Use .asc to export to an ESRI ASCII Grid file.

  3. Enable the Open output file after running algorithm option.

  4. Click on Run to execute the algorithm.

The new file will be saved in the new format and loaded into the map canvas.

Once you are done, click Move to next step to finish the lesson.
