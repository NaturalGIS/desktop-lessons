In the Translate (Convert Format) dialog, do:

  1. For the Input layer choose the dem25 layer.
  2. Type in directly or use the Select... button next to Output file to choose the folder and filename of the file to create. Let's use dem25.asc as filename.

    3. Note: The filename extension is what determines the output's format. If no extension is set, the output file will default to GeoTIFF.

  3. Activate the Load into canvas when finished option.
  4. Click OK to run the tool
  5. Click OK in the progress and cinclusion messages.
  6. Back at the Translate (Convert Format) dialog, click Close.

The new file will be saved in the new format, and loaded into the map canvas.

Once you are done, click Move to next step to finish the lesson.
