In the Open a GDAL Supported Raster Data Source dialog:

  1. Browse to a folder where you saved the map canvas image.
  2. Select the my_map_canvas.png file.
  3. Press Open

You will be prompt to choose the image's Coordinate Reference System. The Save as image tool always uses the current project's CRS. So, for this example, in the Coordinate Reference System Selector choose WGS84 / Pseudo Mercator (EPSG:3857)

The image will open in the map canvas. At first glance, it will look like if you have turned the all the layers back on, but zooming in and out will show that it's in fact an image, that is placed exactly over Wake County location.
